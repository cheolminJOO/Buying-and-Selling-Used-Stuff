## BUSU (Bying Used item & Selling Used item)

Tech Stack
 - Next.js, graphQL,  React-hook-form, TypeScript, CRA, Emotion, Github

주요기능
 - 주소 찾기 ( 카카오 MAP, 카카오 주소찾기 api )
 - 게시판 검색
 - 댓글 무한스크롤
 - 게시판 페이지네이션
 - 장바구니
 - 강아지 open API
